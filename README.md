# Spot On Solutions - Website Redesign

A modern, editorial-style website for Spot On Solutions, a Yorkshire-based creative design agency with 40+ years of experience.

![Spot On Solutions](https://img.shields.io/badge/Status-Complete-brightgreen) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black)

## Overview

A complete redesign of [spoton.org.uk](https://spoton.org.uk), transforming it from a dated WordPress theme into a fast, modern, single-page website that properly showcases the agency's creative expertise.

## Features

- **Editorial Design** - Bold typography with Instrument Serif & DM Sans
- **Custom Cursor** - Animated dot with smooth follower ring
- **Scroll Animations** - Staggered reveal effects throughout
- **Responsive** - Mobile-first design, works on all devices
- **Fast Loading** - No heavy video backgrounds, optimised CSS
- **Accessible** - Semantic HTML, proper heading hierarchy

## Sections

| Section | Description |
|---------|-------------|
| Hero | Bold headline with company stats (40+ years, 500+ projects) |
| About | Company story and collaborative approach |
| Services | NPD, Design, Graphics, Artwork, Adaptive Artwork |
